## Survivors

Connected free quotient:
```tex
BE\simeq BT^2\simeq K(\mathbb Z^2,2),\qquad
H^*(BE;\mathbb F_2)=\mathbb F_2[u_1,u_2],\quad |u_i|=2.
```
Hence
```tex
H^1(BE;\mathbb F_2)=0,\qquad
H^2(BE;\mathbb F_2)=\mathbb F_2u_1\oplus\mathbb F_2u_2,
```
and the connected quotient class is exactly
```tex
\alpha^{E,\mathrm{free}}_{\mathcal C}
=a_1(\mathcal C)u_1+a_2(\mathcal C)u_2.
```
This kills the false `E_2^{1,1}` lane, but it does not trivialize
`a_1,a_2`.

Finite stabilizers:
```tex
E[N]\simeq(\mathbb Z/N)^2.
```
If `N` is odd, transfer gives
```tex
H^{>0}(BE[N];\mathbb F_2)=0.
```
If `2^a || N`, positive-degree mod-2 cohomology is detected on
`(\mathbb Z/2^a)^2`.

For `N=2`:
```tex
H^*(BE[2];\mathbb F_2)=\mathbb F_2[x_1,x_2],\quad |x_i|=1,
```
```tex
H^2(BE[2];\mathbb F_2)
=\mathbb F_2\langle x_1^2,x_1x_2,x_2^2\rangle.
```
The residual is
```tex
\beta^{E,E[2]}_{\mathcal C}
=b_{20}x_1^2+b_{11}x_1x_2+b_{02}x_2^2.
```
With `E[2]=<e_1,e_2>` and `e_3=e_1+e_2`,
```tex
\operatorname{res}_{<e_i>}\beta^{E,E[2]}_{\mathcal C}=r_i\tau_i^2,
```
```tex
r_1=b_{20},\qquad r_2=b_{02},\qquad
r_3=b_{20}+b_{11}+b_{02},
```
so
```tex
(b_{20},b_{11},b_{02})=(r_1,r_1+r_2+r_3,r_2).
```
Thus the `N=2` residual vanishes iff `r_1=r_2=r_3=0`.

For even `N>=4`, `a>=2`:
```tex

=\Lambda(x_1,x_2)\otimes\mathbb F_2[y_1,y_2],
\qquad |x_i|=1,\quad |y_i|=2,\quad x_i^2=0,
```
```tex
H^2(B(\mathbb Z/2^a)^2;\mathbb F_2)
=\mathbb F_2\langle y_1,x_1x_2,y_2\rangle.
```
Write
```tex
\beta^{E,N}_{\mathcal C,(2)}
=A_1^{(N)}y_1+A_{12}^{(N)}x_1x_2+A_2^{(N)}y_2.
```
For `h_i=2^{a-1}g_i`, `i=1,2`, and `h_3=h_1+h_2`,
```tex
\operatorname{res}_{<h_1>}\beta^{E,N}_{\mathcal C,(2)}
=A_1^{(N)}\tau_1^2,\qquad
\operatorname{res}_{<h_2>}\beta^{E,N}_{\mathcal C,(2)}
=A_2^{(N)}\tau_2^2,
```
```tex
\operatorname{res}_{<h_3>}\beta^{E,N}_{\mathcal C,(2)}
=(A_1^{(N)}+A_2^{(N)})\tau_3^2.
```
The mixed coefficient `A_{12}^{(N)}` is invisible to order-two cyclic
restrictions. It is detected by the rank-two quotient
`(\mathbb Z/2^a)^2 -> (\mathbb Z/2)^2`, which pulls
`\bar x_1\bar x_2` back to `x_1x_2`.

## Weyl Transport

```tex

=\left(\theta^{\mathrm{red}}_{\mathcal C},
\theta^{E,\mathrm{free}}_{\mathcal C},
\{\theta^{E,N}_{\mathcal C}\}_{N\ge2}\right).
```
For a type-II simple reflection, failure to transport these
trivializations is the torsor class
```tex

\in
H^1(\mathfrak M^{\mathrm{red}}_{s_{\delta_i}\mathcal C};\mathbb F_2)
\oplus
\bigoplus_{N\ge2}H^1(BE[N];\mathbb F_2).
```

The connected `BE` term contributes no `H^1`, since `H^1(BE;F_2)=0`.
The Weyl-equivariant quotient orientation requires
`\omega_{i,\mathcal C}=0` and Coxeter coherence. Without this, the sign
is only a sign of an ordinary square-root line.
