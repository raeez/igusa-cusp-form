## Proposal

Future `compute/verify_source_fixture.py` should verify certificate
packets under `certificates/`; it should not become a data store. It
consumes an A067-style packet, imports target fixtures only as target
data, and emits `verifier.json` plus `verifier-summary.tex`. It must
never certify target arithmetic or a mock packet as compact source
recognition.

```text
verify_source_fixture(packet_dir, *, window):
  01. packet/preflight gate
      require packet_dir exists; load manifest.yaml; verify hashes;
      require source_kind in {compact_source_candidate, mock_source_fixture,
      target_reference}; fail proof mode unless compact_source_candidate.

  02. schema/scalar/firewall gate
      require exact scalar domain: Z, Q, or declared number field; floats fail.
      source basis ids must be neutral b^X_{R,beta,p,a}.
      forbid target labels e_i,E_ij,u_ij,r,T_i,M_ij,r,w_s outside target_basis
      and A_entries.

  03. window/degrees/target gate
      check S_W = W union -W union {0}; W downward saturated.
      For W_le3 require target rows:
        delta_i: 1|0; a_ij: 10|0; 2delta_i+delta_j: 1|0; tau: 29|93.
      For W_le3, height-four/height-seven terminal rows must be DEFERRED.
      For W_le7, those rows must be CERTIFIED, not deferred.
      Reject signed-only target parity as input to target_basis, PBW, or A.

  04. provenance gate
      every source basis vector has retained stack/stratum, vanishing cycle,
      orientation, TS transport, protected integration, stabilizer, transition.

  05. pair/decomposition coverage gate
      pairs.csv and decompositions.csv enumerate every retained M and D row
      in S_W, including positive, negative, zero, and allowed mixed rows.

  06. matrix construction gate
      load M,D,G and matrix_index; derive or check
        B_{alpha,beta} = M_{alpha,beta} -
          (-1)^{|x||y|} M_{beta,alpha} tau.
      matrices act on column coordinates; tau is ordinary flip.

  07. radical/splitting gate
      for each beta,p:
        K = ker G_beta,p intersect ker G_{-beta,p}^t
        rank K equals nullity([G_beta; G_{-beta}^t])
        P = K direct_sum L; QK=0; Q|_L=I
        restricted quotient pairing is nondegenerate.

  08. NO7/Hopf gate
      verify QB(P tensor K)=0, QB(K tensor P)=0,
      (Q tensor Q)D K=0, and
      M^t G = (G tensor G)D for every retained row.

  09. relation gate
      check Chevalley, real Serre, isotropic orthogonality, super sign,
      Jacobi T_1+T_2+T_3=0, the 27 mixed maps, exponent 3 real strings,
      exponent 5 complementary strings, with window-appropriate deferrals.

  10. comparison-map gate
      each A_{beta,p}: L_{beta,p} -> g_{Delta5,beta,p} is parity-preserving,
      full rank, and intertwines bracket, coproduct, pairing, and Cartan.

  11. generation/surjectivity gate
      finite presentation map pi_W generated by Cartan and supplied simple
      representatives has rank dim L_{beta,p} in every retained block.

  12. no-extra gate
      compute N = ker pi_W and S = span(BK relation rows, GN radical rows).
      pass iff pi_W S=0 and rank N = rank S = rank [N S].

  13. PBW gate
      compare associated gradeds, not Hilbert series alone.
      For W_le3 require tau PBW target ranks:
        d=1: 29|93; d=2: 30|0; d=3: 1|0.
      every recorded gr_map_matrix_id has exact full rank.

  14. transition/ML gate
      verify transition cocycles, preservation of M,D,B,G,K,Q,A,
      PBW strictness T(F^d U') = T(U') cap F^d U,
      and stable-image rank rank M_munu = rank M_mu0nu = rank[M_munu M_mu0nu].

  15. emit
      write per-gate PASS|FAIL|DEFERRED, exact ranks, residual ids,
      matrix hashes, command line, certificate id, target id, and fail code.
```

## Minimal fixtures

`missing_packet`: no directory; expects `MISSING_SOURCE_FIXTURE`.

`target_reference_wle3`: target degrees/bases only; parser works, proof
mode fails.

`mock_source_wle3_good_toy`: exact small neutral-source packet over `Q`;
may pass algebra plumbing under `--allow-mock`, but final status is
`MOCK_NOT_PROOF`.

`bad_target_label_source`: puts `e_i` or `T_i` in
`basis_provenance.csv`; expects firewall failure.

`bad_float_scalar`: one matrix entry `0.1`; expects
`NONEXACT_SCALAR`.

`bad_radical`: wrong `K_entries`; expects `BAD_RADICAL_KERNEL`.

`bad_no7`: nonzero `QB(P tensor K)` residual; expects NO7 failure.

`bad_no_extra`: same visible dimensions but enlarged kernel; expects
`NO_EXTRA_KERNEL_FAIL`.

`bad_pbw`: correct root dimensions but deficient `gr_map_matrix_id`;
expects `PBW_GR_FAIL`.

`wle3_terminal_overclaim` and `wle7_deferred_terminal`: enforce the
supremum rule that `W_le3` is not relation closure and `W_le7` must
close the terminal rows.
