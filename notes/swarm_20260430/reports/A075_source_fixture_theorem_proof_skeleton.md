this form:

```tex

```


not a source construction.


```tex
\rank M_{\mu\nu}
=\rank M_{\mu_0\nu}
=\rank[\,M_{\mu\nu}\ M_{\mu_0\nu}\,].
```

This is what lets kernels, radicals, PBW gradeds, and parity pieces
commute with inverse limit.

## Algebraic Lemma Order

Finite provenance lemma -> Hall matrix lemma -> radical descent lemma ->
relation containment lemma -> comparison-map lemma ->
generation/surjectivity lemma -> no-extra/injectivity lemma ->
PBW associated-graded lemma -> strict transition/ML lemma.

Surjectivity is the rank statement

```tex
\rank\pi_{R,W,\beta,\bar p}=\dim L_{\beta,\bar p}
```

for every retained block, with `\pi` generated by Cartan and supplied
simple representatives through reduced source brackets.

Injectivity is exactly the no-extra kernel equality above.

PBW is not implied by radical descent. It is the independent
associated-graded rank check.
